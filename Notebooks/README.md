# NLP Notebooks

This directory contains Jupyter notebooks for Natural Language Processing experiments and tutorials.

## Contents

- `nlp_notebook.ipynb` - NLP experiments and analysis

## Getting Started

To run these notebooks:

1. Install required dependencies: `pip install -r ../requirements.txt`
2. Launch Jupyter: `jupyter notebook`
3. Open the notebook files to explore the NLP experiments

## Requirements

See the main `requirements.txt` in the parent directory for dependency details.
